Student Management System

Overview

A Java based Student Management System that allows users to manage student records through a menu driven console application. The project demonstrates Object Oriented Programming, CRUD operations, and file handling for data persistence.

Features

- Add student records
- View all students
- Search students by ID
- Update student information
- Delete student records
- Save data to a file
- Automatically load saved data on startup

Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayList Collections
- File Handling

Project Structure

- Student.java – Student model class
- StudentManager.java – Business logic and CRUD operations
- Main.java – Menu-driven application
- students.txt – Stores student data

How to Run

1. Compile the Java files.
2. Run Main.java.
3. Use the menu options to manage student records.

Skills Demonstrated

- Java Programming
- Data Structures
- File I/O Operations
- Problem Solving
- Software Development Fundamentals
